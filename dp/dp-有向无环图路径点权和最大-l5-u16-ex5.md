利用邻接表建图，然后枚举计算每一个顶点作为起点的所有路径最大和。实现时可以通过 dfs 遍历该顶点作为起点的所有路径。最后打擂台找到整张图的最大路径和并输出。

状态 dp[i]：i 号顶点为起点的所有路径最大和

状态转移方程
```cpp
// dp[i]：i 号顶点为起点的所有路径最大和

dp[i] = a[i];
for (int j = 0; j < v[i].size(); j++) {
    dp[i] = max(dp[i], a[i] + dfs(v[i][i]));
}
ans = max{dp[i]} // 最终结果
```
