## 滑动窗口例题：区间取数

一个数组，成员都是正整数。要从中选择一个连续子数组，其所有元素乘积的2进制末尾至少有k个0。求这个连续子数组的最短长度。

第一行输入两个正整数，n和k。第二行输入n个正整数。

### 解题思路

数的二进制末尾存在k个0，也就是说，它是 2^k 的倍数。对数组的每个成员，求出其因数2的个数，放到数组a[] 里。对该数组用滑动窗口求解。

```cpp
#include <vector>
#include <algorithm>
#include <climits>
using namespace std;

int num_of_2s(int num) {
    int c = 0;
    while (num % 2 == 0) {
        num /= 2;
        c++;
    }
    return c;
}

int main() {
    int n, k;
    scanf("%d%d", &n, &k);

    vector<int> a(n); // a[i]: 第 i 个数含多少个 2（2的多少次方）
    for (int i = 0; i < n; i++) {
        int num;
        scanf("%d", &num);
        a[i] = num_of_2s(num);
    }

    int ans = INT_MAX;
    int cnt = 0; // window 里共含多少个2（2的多少次方）

    // 用 for 循环控制 right 每次右移 1
    // 每次 right 右移 1（也就是每次进入循环体），cnt 增加。
    //    如果 cnt 没达到 k，说明窗口还不够大，进入下一次循环（以便再次右移 right）
    //    如果 cnt 达到 k，先做统计工作。持续右移 left，直到 cnt 再次达不到 k
    // 注意初值的选择：left = 0, right 循环初值也设置为 left 的初值。
    // 每次加 a[right]，从来不加 a[left]。防止 left、right 加重复。 
    for (int left = 0, right = 0; right < n; right++) {
        cnt += a[right];
        if (cnt >= k) {
            while (cnt >= k) {
                cnt -= a[left];
                left++;
            }
            // 满足 cnt >= k 的窗口范围是 [l-1, r] 闭区间，长度为 r - (l-1) + 1
            printf("[%d ... %d], len:%d\n", left - 1, right, right - left + 2);
            ans = min(ans, right - left + 2);
        }
    }
    if (ans == INT_MAX) {
        ans = -1;
    }
    printf("%d\n", ans);
    return 0;
}
```
