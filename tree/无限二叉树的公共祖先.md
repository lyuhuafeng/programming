# 无限二叉树的公共祖先

https://oj.youdao.com/course/37/282/1#/1/14225

无限二叉树，就是完全二叉树。从 1 开始，则 i 的父节点为 `int(i/2)`。

思路巧妙。

[l5-u13-1-common-ancestor-unlimited.cpp](code/l5-u13-1-common-ancestor-unlimited.cpp)

```cpp
int main() {
    long a, b;
    scanf("%ld%ld", &a, &b);
    while (a != b) {
        if (a > b) {
            a /= 2;
        } else {
            b /= 2;
        }
    }
    printf("%ld\n", a);
    return 0;
}
```

核心代码优化：

```cpp
    while (a != b) {
        a > b ? a /= 2 : b /= 2;
    }
```
